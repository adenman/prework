Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML element — a unit of content in an HTML document formed by HTML tags and the text or media it contains.
  • Opening Tag — the element name used to start an HTML element. The tag type is surrounded by opening and closing angle brackets.
  • Content — The information (text or other elements) contained between the opening and closing tags of an HTML element.
  • Closing tag — the second HTML tag used to end an HTML element. Closing tags have a forward slash (/) inside of them, directly after the left angle bracket.

CSS

  • CSS is the language that provides style to the content of an HTML page. This includes colors, fonts, positioning, and layout.

Git

  • Git is a program that allows us to create and navigate directories within our machine, create and navigate branches, enter a specific project in VS, and much more.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • code .: opens the VS project that you are currently in on the Git terminal
  • cd "": is a command used to switch directories
  • Git add -A: A command to commit all files in the working branch.
  • git commit -m: is used to associate a message with our commit.

JavaScript

  • JavaScript is a language that gives the website the ability to respond to browser events. These events happen in real time and can be triggered by a wide variety of user interactions
  • Functions allow us to write a chunk of code once that can be reused over and over.Events allow JavaScript to respond to user behaviors, like the user hovering their mouse over an HTML element or resizing their window. When an event is fired, a function is executed.
  • The JavaScript code is between the "" tags. This tag alerts the browser that the page contains JavaScript and separates the JavaScript code from the HTML.
